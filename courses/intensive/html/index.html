HTML

Introductionarrow_right

Les objectifs de ce cours sont :

  • Comprendre ce qu'est le langage HTML, connaître rapidement son historique.
  • Comprendre à quoi il va nous servir.
  • Comprendre ce qu'est un navigateur Web.
  • Savoir créer, modifier un fichier HTML.
  • Savoir afficher une page HTML dans un navigateur.
Bases de HTMLarrow_right

Les objectifs de ce cours sont :

  • Connaître la structure d'une page HTML (doctype, head, body, html).
  • Connaître les deux types de balises (classiques et auto-fermantes).
  • Connaître les balises simples HTML.
  • Savoir commenter son code.
Les attributsarrow_right

Les objectifs de ce cours sont :

  • Connaître le fonctionnement des attributs HTML.
  • Connaître les principaux attributs HTML.
  • Connaître les attributs d'identification (class et id).
  • Pratique : Créer un menu "liste à puces".
HTML5 (Audio et Vidéo), IFramesarrow_right

Les objectifs de ce cours sont :

  • Connaître les principales balises pour intégrer des médias.
  • Connaître les principaux attributs de ces balises.
  • Connaître le principe d'une IFrame.
  • Savoir intégrer des medias dans une page HTML.
  • Savoir intégrer du contenu externe dans une page HTML.
Les métadonnées et le réferencementarrow_right

Les objectifs de ce cours sont :

  • Connaître ce qu'est une méta-donnée.
  • Connaître les principales balises et attributs pour mettre en place les méta-données d'une page HTML.
  • Connaître le principe d'encodage des caractères.
  • Savoir mettre en place les principales balises du head d'une page HTML.
  • Connaître les bases du référencement.
Structurer son contenuarrow_right

Les objectifs de ce cours sont :

  • Connaître les éléments HTML structurants.
  • Comprendre leur différence sémantique.
  • Savoir les utiliser dans des cas différents.
  • Connaître les balises et attributs des tableaux en HTML.
  • Savoir créer une page HTML et structurer son contenu.
Les Formulairesarrow_right

Les objectifs de ce cours sont :

  • Connaître le principe des formulaire et des requêtes qui sont génrées par ces derniers.
  • Connaître les différentes balises et attributs nécessaires à la création de formulaires.
  • Connaître le principe de la balise "label"
  • Savoir créer un formulaire HTML.
