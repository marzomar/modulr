Introductionarrow_right

Les objectifs de ce cours sont :

  • Connaître les différents type de langage de programmation.
  • Savoir ce qu'est JavaScript et son historique.
  • Connaître la syntaxe de base de JavaScript.
  • Savoir déclarer une variable.
  • Connaître les types de base
Les structures de contrôlearrow_right

Les objectifs de ce cours sont :

  • Savoir ce qu'est une structure de contrôle
  • Savoir utiliser les opérateurs logiques.
  • Savoir utiliser des structures conditionnelles.
  • Savoir utiliser les boucles
Les fonctionsarrow_right

Les objectifs de ce cours sont :

  • Savoir ce qu'est une fonction.
  • Savoir déclarer et appeler une fonction.
  • Savoir ce qu'est une Exception.
  • Savoir lever et gérer une Exception.
  • Connaître les notions de scope, fonction anonyme et callback.
Objets et Tableauxarrow_right

Les objectifs de ce cours sont :

  • Savoir ce qu'est un Objet.
  • Savor déclarer et manipuler les propriétés d'un Objet.
  • Savoir ce qu'est une méthode, et savoir les manipuler.
  • Savoir ce qu'est un tableau.
  • Savor déclarer et manipuler un tableau.
  • Savor utiliser d'autres types d'objet.
  • Comprendre ce qu'est une API et savoir les manipuler.
Le DOMarrow_right

Les objectifs de ce cours sont :

  • Savoir ce qu'est le DOM.
  • Connaître les principales API natives permettant d'utiliser les parties globales du DOM.
  • Connaître les principaux outils de "debug" fournis par les navigateurs.
  • Connaître les principales API natives permettant de manipuler les éléments HTML.
  • Comprendre le principe des événements liés au DOM (callback, contexte).
  • Savoir récupérer et manipuler des éléments HTML en javascript.
